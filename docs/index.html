Theme Provider - <mx-theme>

The <mx-theme> component provides theming capabilities to all Meridian components. Supply a primary color and it automatically generates a complete 10-shade palette.

Default Theme (Ant Design Blue)

Primary color: --mx-color-primary = #1890ff 

<mx-theme>
  <button class="demo-button">Primary Button</button>
</mx-theme>

Custom Green Theme

Primary color: --mx-color-primary = #52c41a 

<mx-theme primary-color="#52c41a">
  <button class="demo-button">Primary Button</button>
</mx-theme>

Custom Purple Theme

Primary color: --mx-color-primary = #722ed1 

<mx-theme primary-color="#722ed1">
  <button class="demo-button">Primary Button</button>
</mx-theme>

Custom Orange Theme

Primary color: --mx-color-primary = #fa8c16 

<mx-theme primary-color="#fa8c16">
  <button class="demo-button">Primary Button</button>
</mx-theme>

Installation

npm install @meridian-ux/components
# or
yarn add @meridian-ux/components

Usage

import '@meridian-ux/components/mx-theme';

// In your HTML
<mx-theme primary-color="#1890ff">
  <!-- Your components here -->
</mx-theme>

Semantic Aliases

  • --mx-color-primary - Base primary color (shade 6)
  • --mx-color-primary-hover - Hover state (shade 5)
  • --mx-color-primary-active - Active state (shade 7)
  • --mx-color-primary-bg - Background (shade 1)
  • --mx-color-primary-bg-hover - Background hover (shade 2)
  • --mx-color-primary-border - Border (shade 3)
  • --mx-color-primary-border-hover - Border hover (shade 4)
