Getting Started

Learn how to install and use Meridian UX in your project

Installation

Install Meridian UX via npm or yarn:

npm install @meridian-ux/components

Or with yarn:

yarn add @meridian-ux/components

Basic Usage

Import All Components

The simplest way to use Meridian UX is to import all components at once:

// Import all components
import '@meridian-ux/components';

// Now all mx-* components are available

Import Specific Components

For better tree-shaking and smaller bundle sizes, import only the components you need:

// Import specific components
import '@meridian-ux/components/mx-button';
import '@meridian-ux/components/mx-input';
import '@meridian-ux/components/mx-card';

Use in HTML

Once imported, use the components like any other HTML element:

<mx-button type="primary">Click Me</mx-button>

<mx-input 
  placeholder="Enter your name"
  label="Name"
></mx-input>

<mx-card title="Welcome">
  <p>This is a card component</p>
</mx-card>

Framework Integration

React

import '@meridian-ux/components';

function App() {
  return (
    <div>
      <mx-button type="primary">React Button</mx-button>
    </div>
  );
}

Vue

// main.js
import '@meridian-ux/components';

// In your component
<template>
  <mx-button type="primary">Vue Button</mx-button>
</template>

Angular

// app.module.ts
import { CUSTOM_ELEMENTS_SCHEMA } from '@angular/core';
import '@meridian-ux/components';

@NgModule({
  schemas: [CUSTOM_ELEMENTS_SCHEMA]
})
export class AppModule { }

Svelte

// In your component
<script>
  import '@meridian-ux/components';
</script>

<mx-button type="primary">Svelte Button</mx-button>

Vanilla JavaScript

<!DOCTYPE html>
<html>
<head>
  <script type="module">
    import '@meridian-ux/components';
  </script>
</head>
<body>
  <mx-button type="primary">Click Me</mx-button>
</body>
</html>

TypeScript Support

Meridian UX is built with TypeScript and provides full type definitions. TypeScript will automatically pick up the types when you import components.

import '@meridian-ux/components/mx-button';

// TypeScript knows about MxButton and its properties
const button = document.querySelector('mx-button');
button.type = 'primary';
button.size = 'large';
button.disabled = true;

Customization

Meridian UX uses CSS variables for theming. You can customize the design by overriding these variables:

:root {
  --mx-primary-color: #1677ff;
  --mx-font-family: 'Your Custom Font', sans-serif;
  --mx-border-radius: 8px;
}

/* Dark mode */
[data-theme="dark"] {
  --mx-primary-color: #4096ff;
}

See the Theming Guide for more details.

Browser Support

Meridian UX supports all modern browsers:

  • Chrome/Edge (latest 2 versions)
  • Firefox (latest 2 versions)
  • Safari (latest 2 versions)
  • iOS Safari (latest 2 versions)
  • Android Chrome (latest version)

Web Components are natively supported in all modern browsers. No polyfills required.
