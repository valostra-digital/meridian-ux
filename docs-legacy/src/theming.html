Theming

Customize Meridian UX to match your brand

CSS Variables

Meridian UX uses CSS custom properties (variables) for theming. Override these variables to customize the appearance:

:root {
  /* Primary Colors */
  --mx-primary-color: #1677ff;
  --mx-primary-hover: #4096ff;
  --mx-primary-active: #0958d9;
  
  /* Typography */
  --mx-font-family: -apple-system, sans-serif;
  --mx-font-size-base: 14px;
  --mx-line-height-base: 1.5715;
  
  /* Spacing */
  --mx-spacing-xs: 4px;
  --mx-spacing-sm: 8px;
  --mx-spacing-md: 16px;
  --mx-spacing-lg: 24px;
  --mx-spacing-xl: 32px;
  
  /* Border Radius */
  --mx-border-radius-base: 6px;
  --mx-border-radius-sm: 4px;
  --mx-border-radius-lg: 8px;
}

Dark Mode

Dark mode is supported out of the box. Toggle the theme using the button in the header or programmatically:

// Set dark mode
document.documentElement.setAttribute('data-theme', 'dark');

// Set light mode
document.documentElement.setAttribute('data-theme', 'light');

// Toggle theme
const current = document.documentElement.getAttribute('data-theme');
const next = current === 'dark' ? 'light' : 'dark';
document.documentElement.setAttribute('data-theme', next);

Define dark mode colors:

[data-theme="dark"] {
  --mx-primary-color: #4096ff;
  --mx-bg-base: #141414;
  --mx-text-primary: rgba(255, 255, 255, 0.85);
}

Custom Theme Example

Create a custom theme by overriding the CSS variables:

/* Custom Purple Theme */
:root {
  --mx-primary-color: #722ed1;
  --mx-primary-hover: #9254de;
  --mx-primary-active: #531dab;
  --mx-primary-bg: #f9f0ff;
  --mx-primary-border: #d3adf7;
  
  --mx-border-radius-base: 12px;
  --mx-font-family: 'Inter', sans-serif;
}

[data-theme="dark"] {
  --mx-primary-color: #9254de;
  --mx-primary-bg: #1a0033;
  --mx-primary-border: #391085;
}

Component-Specific Styling

Target specific components using CSS:

/* Style all buttons */
mx-button {
  --mx-button-height: 40px;
  font-weight: 600;
}

/* Style specific button */
mx-button.custom {
  --mx-primary-color: #52c41a;
}

/* Style with part attribute (if exposed) */
mx-button::part(button) {
  border-radius: 20px;
}

Best Practices

  • Define your theme variables in a central CSS file
  • Use consistent spacing values from the design system
  • Test your theme in both light and dark modes
  • Ensure sufficient color contrast for accessibility
  • Use relative units (rem, em) instead of fixed pixels when possible
