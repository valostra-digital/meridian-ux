Radio mx-radio

Radio component for single selection. Option 1 Option 2 Option 3 Disabled

Examples

Basic Usage
Example Radio Content
Basic Radio component with default settings.
<mx-radio>
  Example Radio Content
</mx-radio>
Advanced Usage
Advanced Radio
Radio with additional options and customization.
<mx-radio checked disabled value="''" name="''">
  Advanced Radio
</mx-radio>

API Reference

Properties

Property Type Default Description
checked boolean false Whether radio is checked
disabled boolean false Disabled state
value string '' Value of radio
name string '' Name for radio group

Events

Event Description
change Fired when checked state changes

Slots

Slot Description
(default) Label content
