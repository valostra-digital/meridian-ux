Typography mx-typography

Typography component for text and headings with semantic styling. H1 Title H2 Title Secondary text Success text Warning text Danger text Bold text Italic text Underlined text Deleted text Code text Marked text Ctrl + C This is a paragraph of text.

Examples

Basic Usage
Example Typography Content
Basic Typography component with default settings.
<mx-typography>
  Example Typography Content
</mx-typography>
Variants
Type: default Type: primary Type: success Type: warning Type: error Type: info
Different variants of the Typography component.
<mx-typography type="default">Type: default</mx-typography>
<mx-typography type="primary">Type: primary</mx-typography>
<mx-typography type="success">Type: success</mx-typography>
<mx-typography type="warning">Type: warning</mx-typography>
<mx-typography type="error">Type: error</mx-typography>
<mx-typography type="info">Type: info</mx-typography>
Advanced Usage
Advanced Typography
Typography with additional options and customization.
<mx-typography level="1" disabled>
  Advanced Typography
</mx-typography>

API Reference

Properties

Property Type Default Description
variant TypographyVariant 'text' Component variant: title, text, paragraph
level TitleLevel 1 Title level (1-5, only for variant="title")
type TypographyType - Semantic type: secondary, success, warning, danger
disabled boolean false Disabled state
mark boolean false Highlighted text
code boolean false Code style
keyboard boolean false Keyboard shortcut style
underline boolean false Underlined text
delete boolean false Strikethrough text
strong boolean false Bold text
italic boolean false Italic text

Slots

Slot Description
(default) Text content
